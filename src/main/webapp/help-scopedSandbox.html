Specify attributes for the Sandbox scope. Matching subprojects and/or members will be moved from the Integrity Lifecycle Manager server to the Sandbox directory when the Sandbox is created or resynchronized. The following are examples of some attributes for Sandbox scope:
  • type:text
  • name:Image1.jpg
  • anyrevlabellike:Label2
  • memberrevlabellike:MyLabel
  • subproject:#sub1#sub1-0#sub1-0-0
  • path:*sub1-0-1
  • memberrevlabellike:QQQQ && name:mbr-1-0-0-0.txt (Using '&&' denotes logical AND scope functionality)
